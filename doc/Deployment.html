Cambiar al directorio público

cd /directorio/donde/esta/la/web

Subir el archivo install.php

Puedes utilizar scp o pegarlo en un editor en una consola ssh.

Ejecutar instalador y seguir las instrucciones

php install.php

En OVH es:

php.ORIG.5_4 -n install.php

Para cada nueva actualización:

git push develop nombre-de-tu-rama:master

*. Cada nueva actualizaciónn repetir el paso 5
